# Web de Clima

Proyecto frontend MVP para una aplicación de clima.

## Estructura
- `index.html` — Página principal con las localidades.
- `detalle.html` — Página de detalle que muestra información ampliada y pronóstico semanal.
- `css/styles.css` — Estilos adicionales.
- `js/app.js` — Lógica para mostrar la página en index.html.
- `js/detalle.js` — Lógica para mostrar el detalle y pronóstico en detalle.html.

## Como ejecutar
1. Abrir `index.html` en un navegador
2. Hacer click en "Ver detalle" para ir a la página de detalle de la localidad especificada.
3. Para volver a la página de inicio, presionar volver o inicio en la página de detalle.
